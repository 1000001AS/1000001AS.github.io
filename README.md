Make your changes directly in the browser
Scroll down and click "Commit changes"
Your site will update automatically within ~60 seconds


- How this site is hosted
This site uses GitHub Pages — a free hosting service built into GitHub. There is no cost, no server to manage, and no technical setup required.
To enable GitHub Pages on any repository:

Go to Settings → Pages
Under "Branch", select main
Click Save
